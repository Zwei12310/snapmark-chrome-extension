
# SnapMark - 截图标注工具

一键网页截图、标注、导出。Chrome / Edge 扩展。

## 功能
- 一键截图当前网页
- 矩形框选、箭头、文字标注
- 6 种颜色
- 下载 PNG / 复制到剪贴板
- 历史记录管理（最近 50 条）

## 安装方法（侧载）

1. 下载 `snapmark-chrome-extension.zip` 并解压到一个文件夹
2. 打开 Chrome 或 Edge，地址栏输入 `chrome://extensions`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹
5. 完成，点击工具栏 SnapMark 图标开始使用

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Esc | 关闭编辑器 |
| Ctrl+Z | 撤销上一步标注 |
| Ctrl+S | 下载截图 |
